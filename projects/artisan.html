Site vitrine — artisan

Voir le site internet Repository GitHub

Contexte

Projet de site vitrine destiné à présenter l’activité d’un artisan du bâtiment (maçonnerie, couverture, piscines, taille de pierre), avec une approche volontairement simple : mettre en avant les prestations, rassurer le visiteur et permettre une consultation rapide sur mobile comme sur desktop.

Objectif

Concevoir un site léger, rapide à charger, facile à maintenir et simple à déployer, sans complexité technique inutile.
L’idée était de retenir une solution proportionnée à un besoin concret de visibilité en ligne.

Choix techniques

Le projet repose sur une application React compilée en site statique et déployée sur GitHub Pages.
Ce choix permet un coût d’hébergement très faible, une maintenance réduite et une mise en ligne simple. L’accent a été mis sur la lisibilité, la sobriété de l’interface et la facilité d’évolution.

Technologies

React JavaScript HTML / CSS Node / npm GitHub Pages

Ce que ce projet montre

  • capacité à concevoir une solution simple et adaptée au besoin ;
  • attention portée à la clarté de l’interface et à l’accessibilité ;
  • mise en œuvre d’un déploiement statique léger et maintenable ;
  • volonté de privilégier l’efficacité plutôt qu’une sur-ingénierie inutile.

Lancer en local

npm install
npm start

Accès : http://localhost:3000/guillemot

Déploiement

  • npm run build
  • npm run deploy

Le build est publié sur GitHub Pages, ce qui convient bien à un site vitrine : coûts faibles, exploitation simple et maintenance limitée.
