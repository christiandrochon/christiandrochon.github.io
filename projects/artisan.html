Site vitrine — artisan

Voir le site internet Repository GitHub

Objectif

Site vitrine simple pour un artisan (maçonnerie, couverture, piscines, taille de pierre), conçu pour être rapide à charger, facile à mettre à jour et simple à déployer.

Lancer en local

npm install
npm start

Accès : http://localhost:3000/guillemot

Déploiement

  • npm run build
  • npm run deploy

Le build est publié sur GitHub Pages. Approche adaptée à des sites vitrines : coûts faibles, maintenance simple.
