Multimedia — Recherche d’images par similarité

Repo : github.com/christiandrochon/multimedia

Problème & approche

L’utilisateur soumet une image requête. Le système extrait des descripteurs et retourne les images les plus similaires selon une métrique. L’approche combine descripteurs “classiques” (couleur, niveaux de gris, forme, texte) et descripteurs profonds (embeddings via un CNN type ResNet), avec une évaluation quantitative (précision / rappel).

Fonctionnalités clés

  • Upload d’une image requête via l’interface web
  • Modes de recherche : couleur / forme / texte / deep (ResNet) / combinaison pondérée
  • Résultats classés par score de similarité
  • Évaluation : précision, rappel (vérité terrain)
  • Architecture modulaire pour ajouter facilement de nouveaux descripteurs

Exécution via Docker (démo reproductible)

docker compose -f compose.yaml up -d

Accès

  • Frontend Angular : http://localhost:4200
  • Backend API : http://localhost:8087

Architecture

├── data/

│  ├── indexes/       # Indexs descripteurs

API (exemples)

Recherche par image :

POST /api/search  (multipart/form-data)
- file : image requête
- topK : nombre de résultats (optionnel)
- mode : color | shape | text | deep | combined (optionnel)

Exemple curl :

curl -X POST http://localhost:8080/api/search \
  -F file=@image.jpg \
  -F topK=10 \
  -F mode=combined
